# UK Grid Stress Dashboard

This project studies when regional electricity demand growth begins to collide with constrained transmission and distribution systems.

## Core Question

Where do data centre demand, electrification, and limited grid capacity create the greatest system stress?

## Focus

- regional electricity bottlenecks
- network capacity constraints
- power price pressure
- investment implications

## Objective

To build a structured framework for identifying where physical stress may translate into market and infrastructure risk.
